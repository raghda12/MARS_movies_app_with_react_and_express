## How to Run the App

### Prerequisites
* [Node.js](https://nodejs.org/) installed

### How to Run the Frontend
The frontend is built using React and Vite.

1. Navigate to the frontend project directory:
   ```bash
   cd frontend/movie-app
   ```
2. Install the required dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
   The frontend should now be running (typically accessible at `http://localhost:5173`).

### How to Run the Backend
The backend is built with Express.js.

1. Navigate to the backend directory:
   ```bash
   cd backend
   ```
2. Install the required dependencies (assuming a `package.json` exists):
   ```bash
   npm install
   ```
3. Set up environment variables (e.g., create a `.env` file based on a `.env.example`).
4. Start the server:
   ```bash
   npm run start
   ```
   *(Or `npm run dev` if you are using a tool like nodemon for development).*
